## ⚙️ Spécifications Techniques
* **Core Framework:** Python, NumPy, Numba (`@njit` compiled).
* **Compression:** Simulation de bit-packing chirurgical 4-bit (extraction de nibbles).
* **Anchors:** Calibré sur les constantes physiques (Fer-56: 55.84547, Résonance de Schumann: 7.83 Hz).
* **Security:** Accès verrouillé par la clé maître `0x42`.
